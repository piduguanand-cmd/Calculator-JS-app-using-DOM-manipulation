Calculator Web Application :

Project Overview :

This project is a responsive Calculator Web Application developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations and provides an interactive user interface for calculations. The calculator supports keyboard input, button clicks, percentage calculations, clear functionality, delete functionality, and sound effects for button interactions.

Features :

- Addition, subtraction, multiplication, and division
- Percentage calculations
- Clear (C) button
- Delete (DEL) button
- Keyboard support
- Responsive design
- Interactive button click sounds
- Real-time display updates

Technologies Used :

- HTML5
- CSS3
- JavaScript (ES6)

Project Structure :

calculator/
│
├── index.html

├── script.js

How to Run :

1. Download or clone the repository.
2. Open the project folder.
3. Open "index.html" in any modern web browser.
4. Start performing calculations.

Functionalities :

- Numeric input using buttons and keyboard.
- Arithmetic operations (+, −, ×, ÷).
- Percentage calculation.
- Delete the last entered character.
- Clear all calculator data.
- Evaluate expressions using the equals (=) button.
- Sound feedback on button clicks.
